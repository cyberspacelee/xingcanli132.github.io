eval 函数

eval() 函数十分强大 —— 将字符串 当成 有效的表达式 来求值 并 返回计算结果

案例 - 计算器

需求

  1. 提示用户输入一个 加减乘除混合运算
  2. 返回计算结果

不要滥用 eval

在开发时千万不要使用 eval 直接转换 input 的结果

  • 等价代码
  • 执行成功，返回 0
  • 执行失败，返回错误信息
